# Roadmap

## v0.1 — local vertical slice

- encrypted append-only events and deterministic projection;
- Codex, Claude Code, Cursor, Antigravity, and generic adapters;
- typed temporal multigraph, Graphify import, Semble search, and bounded traversal;
- REST, WebSocket, MCP, CLI, and interactive 3D+time UI;
- repo-local marketplace package and reproducible demo fixtures.

## v0.2 — trustworthy daily use

- background projector/index worker and incremental snapshots;
- review inbox for inferred edges and merge suggestions;
- consent dashboard, retention/deletion verification, encrypted backup/restore;
- benchmarks at 100k nodes/1m assertions and accessibility hardening;
- versioned compatibility matrix for each host adapter.

## v0.3 — hosted graph sync

- OAuth, verified tenant isolation, graph-only sync, conflict resolution, quotas, and billing;
- signed releases, SBOM/provenance, hosted MCP, public policies, and marketplace submission;
- optional team sharing only after permission inheritance and deletion semantics pass an external review.

## Later, only with evidence

A2A delegation, mobile/VR, richer Graphify pipelines, team RBAC, and opt-in content sync are hypotheses. They are not v1 dependencies and should be promoted only after usage data shows that they improve continuity without weakening privacy or reliability.
